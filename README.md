# Skill Gap Analyzer

## 📌 Description
A Django-based web application that analyzes a student's current skills
and compares them with industry requirements.

## 🛠 Tech Stack
- Python
- Django
- HTML, CSS, Bootstrap
- SQLite

## ⚙ Installation
1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Run server

## 🚀 Features
- User authentication
- Skill analysis
- Learning roadmap
- Progress tracking
